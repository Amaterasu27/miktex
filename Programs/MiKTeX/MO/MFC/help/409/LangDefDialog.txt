.topic HIDC_LANG_NAME
The name of the language.

.topic HIDC_SYNONYMS
Comma-separated list of synonyms.

.topic HIDC_INPUT_NAME
Specifies the name of the main input file for the format.

.topic HIDC_HYPHENATION_FILE
Specifies the name of the file which contains the hyphenation patterns.

.topic HIDC_BROWSE
Click this to browse for the hyphenation pattern file.

.topic HIDC_ACTIVE
Specifies whether the hyphenation patterns should be loaded by iniTeX.
