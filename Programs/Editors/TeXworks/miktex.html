TeXworks for MiKTeX

The MiKTeX distribution now includes TeXworks, a simple environment for editing TeX/LaTeX documents written by Jonathan Kew.

The MiKTeX version has the following features:

  • By default, the MiKTeX compiler driver (texify) is used to typeset LaTeX documents.
  • Some spelling dictionaries (*.dict) are included in the standard MiKTeX distribution.
  • TeX/LaTeX related file name extensions (*.tex, *.sty, *.pdf, ...) are associated with TeXworks.
  • An extra About dialog has been added to the Help menu.
  • When sending a message to the TeXworks mailing list (Help->Email to the mailing list), the subject contains the word MiKTeX.
  • The MiKTeX bin directory is automatically added to the program path (Edit->Preferences...->Typesetting).
